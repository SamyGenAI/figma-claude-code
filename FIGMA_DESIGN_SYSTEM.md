## Design Tokens

**Location:** `tailwind.config.js` ‚Üí `theme.extend`

All tokens are consumed via Tailwind utility classes (e.g. `bg-navy`, `font-heading`, `rounded-card`). No separate token file or transformation pipeline.

| Token | Value | Usage |
|---|---|---|
| `navy` | `#092c69` | Section headers, footer, title text |
| `cream` | `#fffceb` | Page background |
| `blue-light` | `rgba(180,234,255,0.7)` | Cowork icon badges |

| Token | Font | Weights loaded |
|---|---|---|

## Asset Management

**All Figma-sourced assets are remote URLs**, stored in `src/assets/figma-assets.js` as named ES module exports.

```js
// src/assets/figma-assets.js
export const imgClaudeIcon = 'https://www.figma.com/api/mcp/asset/<uuid>';
export const iconChat      = 'https://www.figma.com/api/mcp/asset/<uuid>';
```

**Usage:**
```jsx
import { imgClaudeIcon } from '../assets/figma-assets';
<img src={imgClaudeIcon} alt="Claude icon" className="size-[102px]" />
```

> **IMPORTANT ‚Äî Asset expiry:** Figma MCP asset URLs expire after **~7 days**.
> To refresh: call `mcp__figma__get_design_context(nodeId: "1:21", fileKey: "ksbyxTwKodTZJnvgwNhoCn")`
> and update the URLs in `src/assets/figma-assets.js`.

Local static assets: `public/vite.svg` (unused default, can be deleted).
No CDN, no image optimization pipeline.

## Styling Approach

**Methodology:** Tailwind CSS utility classes ‚Äî no CSS Modules, no Styled Components, no SASS.

**Global styles:** `src/index.css`
- Tailwind directives (`@tailwind base/components/utilities`)
- Google Fonts import
- `box-sizing: border-box` reset
- `.grid-bg` custom class: cream background + 40√ó40px light-grey CSS grid pattern

```css
.grid-bg {
  background-color: #fffceb;
  background-image:
    linear-gradient(rgba(0,0,0,0.07) 1px, transparent 1px),
    linear-gradient(90deg, rgba(0,0,0,0.07) 1px, transparent 1px);
  background-size: 40px 40px;
}
```

**Responsive design:** None ‚Äî this is a fixed-size infographic (`1080√ó1350px`). Arbitrary Tailwind values (`text-[72px]`, `left-[59px]`) are used to precisely match Figma pixel coordinates.
